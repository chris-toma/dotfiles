| Ctrl+N | Go to class | Move caret down |
| Ctrl+E | Recent files | Move to end of line |
| Ctrl+D | Duplicate line | Delete character |
| Ctrl+V | Paste | Page down |
| Ctrl+Y | Delete line | Emacs yank |
| Ctrl+K | Commit | Delete to end of line |
| Ctrl+T | Update project | Transpose characters |

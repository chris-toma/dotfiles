# GoLand Custom Keymaps

Parent keymap: `Mac OS X 10.5+`

## F10 Chord Bindings

| Shortcut | Action |
|---|---|
| `F10, 1` | ActivateProjectToolWindow |
| `F10, a` | Annotate (git blame) |
| `F10, b` | Git.Branches |
| `F10, c` | Coverage |
| `F10, d` | ActivateDatabaseToolWindow |
| `F10, e` | ActivateElectroJunToolWindow |
| `F10, f` | HideAllWindows |
| `F10, h` | Git.Log.Hide.Branches |
| `F10, k` | copilot.git.commit.button |
| `F10, p` | ManageRecentProjects |
| `F10, q` | CloseProject |
| `F10, r` | ChooseDebugConfiguration |
| `F10, s` | Git.Stash |
| `F10, space` | SearchEverywhere |
| `F10, t` | ActivateTerminalToolWindow |
| `F10, u` | Git.Unstash |
| `F10, v` | VimPluginToggle |
| `F10, x` | Console.TableResult.Transpose |
| `F10, z` | ToggleZenMode |
